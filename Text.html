Font Style

css dapat mengatur style dari teks yang ada pada halaman web yang di buat terdapat beberapa property yang dapat digunakan untuk mengatur style teks yaitu sebagai berikut.

  • tekt-align: digunakan untuk mengatur perataan teks, value yang dapat digunakan yaitu left, center, right
  • color: digunakan untuk memberikan warna pada teks
  • letter-spacing: mengatur jarak antar karakter
  • world-spacing: mengatur jarak antar kata
